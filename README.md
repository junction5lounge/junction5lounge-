## 🚀 Deployment

### Run on Replit
1. Import this repo into Replit (Node.js template)
2. Hit **Run**
3. Site goes live automatically on port 3000

### Run Locally
```bash
node server.js
```
Then open `http://localhost:3000` in your browser.

## 💳 Paystack Integration

Payments are handled via **Paystack** for:
- Table reservation deposits (GH₵20/guest + celebration packages)
- Food order checkout

To activate payments, open `index.html` and replace:
```javascript
PK: 'pk_test_REPLACE_WITH_REAL_PAYSTACK_PUBLIC_KEY'
```
With the real Junction 5 Lounge **Paystack Public Key** from:
> paystack.com → Settings → API Keys → Public Key
